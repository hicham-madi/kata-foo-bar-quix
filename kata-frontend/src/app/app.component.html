Kata FOO BAR QUIX

Énoncé du Kata

Ecrire un algorithme qui transforme une nombre (compris entre 0 et 100) en une chaîne de characters. Cet algorithme doit suivre les règles suivantes :

  • Si le nombre est divisible par 3 ou contient 3, la chaîne doit contenir "FOO".
  • Si le nombre est divisible par 5 ou contient 5, la chaîne doit contenir "BAR".
  • Si le nombre contient 7, la chaîne doit contenir "QUIX".
  • La règle "divisible par" est prioritaire sur "contient".
  • Le contenu est analysé de gauche à droite.
  • Si aucune règle n'est vérifiée, retourner le nombre en entrée sous forme de chaîne.

Input Output
1"1"
3"FOOFOO"
5"BARBAR"
7"QUIX"
9"FOO"
51"FOOBAR"
53"BARFOO"
33"FOOFOOFOO"
15"FOOBARBAR"
